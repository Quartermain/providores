C) About the Extension - top

This is a 4 level category module that can be displayed on the left or right side. It has 1 main category and 3 sub categories and supports both LTR and RTL languages.

  • All features are controlled in the Admin.
  • Display on left or right side, enable or disable product counts on all categories, add to any page, enable or disable and select sort order.
  • For the box content you can select the background color, padding, border size, border color and border style.
  • For the fonts you can select the type, size, weight, color, hover color, active color, active hover color and padding for the left or right edges.
  • For the menu you can select the width, height, margin, background color, hover background color, active background color, active hover background color, border size, color and style and the subs top border size, color and style.
  • Has 4 levels, 1 main category and 3 sub categories. Supports LTR and RTL languages.

D) Installation - top

MAKE SURE TO BACKUP YOUR FILES BEFORE CONTINUING. WE ARE NOT RESPONSIBLE FOR ANY DAMAGES.

Step 1 - Upload

Upload the contents of the "upload" folder which are the folders called "admin" and "catalog" to the folder that OpenCart is installed in. These are new files and no core OpenCart files are overwritten.

Step 2 - Set Module Permissions

  1. Login to your Administration panel.
  2. Select "System > Users > User Groups".
  3. Edit the "Top Administrator" group.
  4. Make sure "module/promenu" is checked in both the Access Permission and Modify Permission lists.
  5. Save.

E) Configuration - top

Enable the extension by going to "Extensions > Modules > Pro Category Menu", click "Install" and then "Edit" and set the settings you would like for the module. Click Save.

If you want to customize the CSS any further than what the settings for it provide it is located in the .tpl files. Note that our free support does not include teaching you how to do modifications to these files and no free support will be given for them.
